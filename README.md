A sample Discord bot, to start make your bot.

# TO-START

## Download

clone project :

```
 $ git clone https://github.com/htprogras/starter-ARC.git myBot
```

npm install module :

```
 $ npm install --save
```

## start bot

update file : [ settings.js ]

```
 "token": "YOUR TOKEN"
```

start your project :

```
 $ npm start
```
